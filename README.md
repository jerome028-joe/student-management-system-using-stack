# Student Stack Management System

## Description
The Student Stack Management System is a web-based application developed using HTML, CSS, and JavaScript. It uses the Stack data structure to manage student names and performs Push, Pop, Display, and Top operations following the LIFO (Last In First Out) principle.

## Features
- Add Student (Push)
- Remove Student (Pop)
- Display Stack
- Show Top Student
- User-friendly Interface

## Technologies Used
- HTML
- CSS
- JavaScript

## Data Structure
- Stack (LIFO - Last In First Out)
